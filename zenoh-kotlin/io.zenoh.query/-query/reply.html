Reply success to the remote Query.

A query can not be replied more than once. After the reply is performed, the query is considered to be no more valid and further attempts to reply to it will fail.

Parameters

keyExpr

Key expression to reply to. This parameter must not be necessarily the same as the key expression from the Query, however it must intersect with the query key.

payload

The payload with the reply information.

encoding

Encoding of the payload.

qos

The QoS for the reply.

timestamp

Optional timestamp for the reply.

attachment

Optional attachment for the reply.
