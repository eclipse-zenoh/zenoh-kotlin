Attachment

An attachment consists of a list of non-unique ordered key value pairs, where keys are UTF-8 Strings and the values are bytes. Inserting at the same key multiple times leads to both values being transmitted for that key.

Attachments can be added to a message sent through Zenoh while performing puts, queries and replies.

Using attachments will result in performance loss.
