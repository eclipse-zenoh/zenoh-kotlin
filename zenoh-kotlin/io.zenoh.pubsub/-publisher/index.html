Publisher

A Zenoh Publisher.

A publisher is automatically dropped when using it with the 'try-with-resources' statement (i.e. 'use' in Kotlin). The session from which it was declared will also keep a reference to it and undeclare it once the session is closed.

Example of a publisher declaration:

val keyExpr = "demo/kotlin/greeting".intoKeyExpr().getOrThrow()
Zenoh.open(Config.default()).onSuccess {
    it.use { session ->
        session
            .declarePublisher(keyExpr)
            .onSuccess { pub ->
                var i = 0
                while (true) {
                    pub.put(ZBytes.from("Hello for the ${i}th time!"))
                    Thread.sleep(1000)
                    i++
                }
            }
    }
}

Lifespan

Internally, the Session from which the Publisher was declared keeps a reference to it, therefore keeping it alive until the session is closed. For the cases where we want to stop the publisher earlier, it's necessary to keep a reference to it in order to undeclare it later.
